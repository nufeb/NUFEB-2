
Balls! Stuff for messing around with growing, dividing and shoving cells.

python balls.py
should run a simple 2d demo, which should only rely on the "pygame" library, which is installed by default with most python installations

python balls3d.py
should run a simple 3d demo. However, this relies on the "panda3d" library being installed, which is not installed by default, and is quite a big and complex package.
